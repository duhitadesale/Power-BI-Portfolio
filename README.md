# 1️⃣ Toy Store KPI Dashboard

## Business Problem

A toy retailer needs a centralized reporting solution to monitor business performance across stores, products, and categories.

Management requires visibility into:

* Revenue performance
* Profitability trends
* Order volume
* Product performance
* Store-level performance

## Dashboard Objectives

* Track key business KPIs
* Monitor sales and profit trends
* Identify top-performing products
* Compare store performance
* Support data-driven business decisions

## Key Features

* Executive KPI Cards
* Sales Performance Tracking
* Profit Analysis
* Product Performance Insights
* Store-Level Analysis
* Interactive Filtering

## Skills Demonstrated

* KPI Dashboard Design
* DAX Measures
* Data Modeling
* Business Intelligence Reporting
* Visual Storytelling

# 2️⃣ LEGO Set Explorer

## Business Problem

Consumers, retailers, and product analysts need a better way to explore LEGO products based on:

* Price
* Age suitability
* Theme popularity
* Product complexity
* Set characteristics

## Dashboard Objectives

* Explore LEGO sets interactively
* Analyze pricing patterns
* Compare themes and categories
* Understand age-group targeting
* Identify trends in LEGO products

## Key Features

* Interactive Set Explorer
* Theme Analysis
* Pricing Insights
* Age Group Analysis
* Product Complexity Metrics
* Dynamic Filtering Experience

## Skills Demonstrated

* Data Exploration
* Consumer Product Analytics
* Dashboard UX Design
* Data Visualization
* DAX Calculations
